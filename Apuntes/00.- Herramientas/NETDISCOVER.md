#Enumeration #RedEnumeration 

| OPCIONES | FUNCION             | +Info                           |
| -------- | ------------------- | ------------------------------- |
| *-i*     | Interfaz de escaneo |                                 |
| *-r*     | Rango a escanear    | Poner la mascara de red ej: /24 |

Para enumerar un rango completo
``` bash
netdiscover -i <Interfaz> -r <rango/mascara>
```
