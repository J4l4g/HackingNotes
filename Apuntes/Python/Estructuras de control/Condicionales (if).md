```ad-info
- Ejecuta código condicionalmente según se cumpla una condición.
- Permite desviar el flujo del programa.
- Muy útil para decisiones y validaciones.
  
#### Usos comunes
- Validar entradas de usuario.
- Tomar decisiones según valores.
- Ejecutar alternativas según condiciones múltiples.
```

**Cómo se usan:**
```python
if condicion:
    # bloque si True
elif otra_condicion:
    # bloque alternativo
else:
    # bloque si ninguna condición es True
```


**Cheatsheet básico:**
```python
x = 10

if x > 0:
    print("Positivo")
elif x == 0:
    print("Cero")
else:
    print("Negativo")

# Operadores de comparación: ==, !=, >, <, >=, <=
# Operadores lógicos: and, or, not
```
